# Casos de Uso

## UC-01: Autenticar Usuário
- **Ator**: Usuário
- **Pré-condição**: Usuário possui cadastro.
- **Fluxo principal**:
  1. Usuário informa e-mail e senha.
  2. Sistema valida as credenciais.
  3. Sistema exibe a página inicial.
- **Pós-condição**: Usuário logado.
